I am a proactive IT professional with a passion for software quality assurance, API testing, and automation. I hold certifications in IT Customer Support and Data Analytics. My focus is on building robust test frameworks that ensure software reliability and performance.

### **Skills**

* **Manual Testing:** Test case design, defect reporting, Jira workflow management.
* **Automation Testing:** Java, Selenium, TestNG, ExtentReports.
* **API Testing:** RestAssured, Postman, Positive & Negative testing.
* **Tools & Platforms:** IntelliJ, Maven, Git/GitHub, Cypress basics.
* **Reporting & CI/CD:** ExtentReports, Maven integration, GitHub repository management.

### **Project: API Automation Portfolio**

**Objective:** Automate user API endpoints, covering both positive and negative tests.

**Tech Stack:** Java, RestAssured, TestNG, ExtentReports.

**Key Highlights:**

* Full test coverage including positive & negative API scenarios.
* Detailed ExtentReports dashboard with color-coded status, request/response logging.
* Ready-to-run on any system with Maven setup.
* Clean code organized with BaseTest, Utils, and separate test packages.

### **Installation & Setup**

1. Clone the repository: `git clone https://github.com/LeratoM99/ApiAutomationFramework.git`
2. Open the project in IntelliJ or your preferred IDE.
3. Run `mvn clean install` to install dependencies.
4. Execute tests with TestNG or via Maven commands.

### **Usage**

* Positive & Negative API tests included.
* ExtentReports automatically generated in `test-output/ExtentReport.html`.
